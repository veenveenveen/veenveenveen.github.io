gitalk配置出现的问题

1. gitalk授权登录后报错403

答：更新版本到 1.7.2 或者 修改配置增加 proxy: https://cors-anywhere.azm.workers.dev/https://github.com/login/oauth/access_token

2. 未找到相关的issues进行评论，请联系@XXX初始化创建。

答：未给文章评论初始化，博主登录 GitHub 账户进行初始化即可，注：每次创建新的文章都需要博主对该文章进行一次issues初始化。

3. Error: Not Found.

答：未能找到仓库 repo，检查一下你的仓库名称是否填写准确（只写名称即可）

4. 后续出现别的问题再进行补充
